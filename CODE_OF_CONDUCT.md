# Code of Conduct

## Our pledge

We want this project to be welcoming to everyone, regardless of experience level, background, or identity.

## Standards

**Do:**
- Be respectful and constructive in discussions.
- Give and accept feedback gracefully.
- Focus on what's best for the project.

**Don't:**
- Use hostile, demeaning, or exclusionary language.
- Harass anyone, publicly or privately.
- Publish others' private information without permission.

## Enforcement

Unacceptable behavior can be reported to hello@martinbrian.com. Reports will be reviewed and addressed promptly. Maintainers who don't uphold the code of conduct may be removed from the project.
